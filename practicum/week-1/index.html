In this practicum you will develop a responsive webpage using Bootstrap. The webpage that you need to develop is a simple verstion of the first page of this website: https://giftsear.ch

What you need to do:

  1. Create a github repository and name it mygiftsearch
  2. Your mygiftsearch web page should have the following Boostrap components:
    1. Bootstrap container
    2. Bootstrap Image
    3. Bootstrap Button
    4. Bootstrap Card
    5. Bootstrap Navigation Bar
  3. You just need to develop one single page. The links in the navigation bar can be all broken links.
  4. Your design should be responsive, meaning it should look good on all devices.
